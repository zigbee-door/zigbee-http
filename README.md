# zigbee-http

## 启动

### 本地环境

```javascript
npm start               //Mac Os X
```

```javascript
npm run start_win       //Windows
```

### hik环境(仍然是开发环境)

```javascript
npm run start_win_dev   //Windows
```

## 功能

| 插件      |     描述 |
| :-------- | :--------|
| mongoose    |   mongodb数据库存储 |
| redis    |   缓存和进程间通信 |
| socket.io    |   websocket通信 |
| bootstrap    |   响应式设计 |
| bootstrap-table    |   丰富的表格形式 |
